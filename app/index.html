При выполнении верстки использовал:

  • CSS препроцессор Sass (SCSS)
  • HTML шаблонизатор Pug
  • Сборщик проекта Gulp (сборка включена в репозиторий)

Сайт разделен на ДВЕ страницы: index.html и post.html (это было сделано из-за конфликта валидации двух одинаковых форм)

Подключал шрифты разными способами (Google fonts и локально)
mixin для шрифтов прописаны в файле по пути sass/_fonts.css

Все анимации написаны вручную без библиотек на css

Форма полностью валидируется, при успешной отправки появится popup, информирующий об успешной отправки

Валидация формы была полностью написана вручную, без сторонних помощников с нуля (прям в этом проекте) на нативном JS

Так же данная верстка полностью адаптивная и кроссбраузерная
